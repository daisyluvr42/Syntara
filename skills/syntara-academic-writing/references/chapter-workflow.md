# Chapter Workflow

Use this reference for a full chapter draft.

## 1. Prepare

Restate the user's requested topic, title, outline, audience, and evidence scope. If the user supplied source prose, extract a style brief before retrieval.

## 2. Plan Retrieval

Turn the outline into section-level questions. Good questions are narrow enough to retrieve evidence:

- mechanism questions
- indication/contraindication questions
- procedure-step questions
- complication-management questions
- comparative-effectiveness questions
- terminology-definition questions

Avoid a single broad query for an entire chapter.

## 3. Retrieve Evidence

For each section:

1. Search Syntara.
2. Read context for high-value hits.
3. Use RAG for one bounded synthesis if needed.
4. Record cite keys and unresolved gaps.

## 4. Draft

Write section by section. Use the user's outline unless it is internally inconsistent; if it is inconsistent, point out the issue before rewriting the structure.

Keep three kinds of sentences distinct:

- Literature-supported claims with cite keys.
- Mechanistic explanation derived from accepted domain logic.
- Experience-based clinical judgment, marked with appropriate caution.

## 5. Revise

Do a final pass for:

- style brief consistency
- section order and transitions
- terminology consistency
- citation coverage
- unsupported claims
- places where more Syntara retrieval is needed

Return the draft first, then a short evidence-gap note.
